# 핵심 기능 목록

## MoSCoW 우선순위

| 레벨 | 의미 |
|------|------|
| **Must Have** | 반드시 구현. 없으면 앱의 핵심 가치가 성립되지 않음 |
| **Should Have** | 강력히 권장. 없으면 사용자 경험이 크게 저하됨 |
| **Could Have** | 있으면 좋음. 시간 여유 시 구현 |
| **Won't Have** | 현재 범위 제외. 향후 버전 고려 |

### Must Have

- [x] 공공 레시피 조회·검색 (필터·정렬 포함)
- [x] 이메일 회원가입 / Google 로그인
- [x] 냉장고 파먹기 (보유 재료 기반 레시피 자동 매칭)
- [x] 레시피 상세 (재료 목록·단계별 조리 순서)
- [x] 비로그인 게스트 모드 진입

### Should Have

- [x] 북마크·좋아요·최근 본 레시피 개인화
- [x] 커뮤니티 게시글·댓글 (카테고리 분류)
- [x] 냉장고 재료 등록 및 유통기한 관리
- [x] 댓글·유통기한 임박 알림
- [x] 카카오 로그인
- [x] 프로필 공개/비공개 설정

### Could Have

- [x] 업적·칭호 시스템 (26가지)
- [x] 조리 전용 모드 (풀스크린·단계별 스와이프)
- [x] 1:1 채팅 (쪽지)
- [x] 레시피 직접 작성·수정·삭제
- [x] 관리자 패널 (대시보드·레시피·게시글·유저 관리)
- [x] 대댓글·댓글 좋아요·익명 댓글

### Won't Have

- [ ] AI 기반 레시피 자동 추천 (재료·취향 분석)
- [ ] 외부 배달 서비스 연동
- [ ] 식단·칼로리 다이어리 기능
- [ ] 인앱 결제·구독 모델
- [ ] 오프라인 레시피 다운로드

| 카테고리 | 세부 기능 |
|----------|----------|
| 인증 | Google 로그인, 카카오 로그인, 이메일 회원가입/로그인 |
| 레시피 | 조회, 필터, 정렬, 검색, 상세보기, 직접 작성, 조리 전용 모드 |
| 개인화 | 북마크, 최근 본 레시피, 좋아요 |
| 냉장고 | 재료 등록, 유통기한 관리, 재료 기반 레시피 매칭 |
| 커뮤니티 | 게시글 작성/조회, 카테고리 분류, 댓글, 좋아요, 1:1 채팅 |

## 1. 인증 기능

- **Google 로그인** — Firebase Auth 기반, 모바일/웹 모두 지원
- **카카오 로그인** — Kakao Flutter SDK, 네이티브 앱 키 연동
- **이메일 회원가입** — 닉네임·이메일 중복 체크, 비밀번호 재설정
- **약관 동의** — 회원가입 시 이용약관·개인정보처리방침 동의 단계 포함
- **최근 로그인 힌트** — 이전에 사용한 로그인 방식 화면에 표시
- **비로그인 게스트 모드** — 로그인 없이도 앱 접근 가능 (북마크·활동 내역·채팅 등 일부 기능은 로그인 유도)

---

## 2. 레시피 기능

### 2-1. 레시피 목록 & 검색
- 국가별 필터: 한식, 서양, 중국, 일본, 동남아시아, 이탈리아, 퓨전
- 유형별 필터: 기타, 반찬 등
- 정렬: 최신순, 인기순(조회수)
- 키워드 검색: 요리명 및 재료명 기반 클라이언트 사이드 필터링

### 2-2. 레시피 상세
- 재료 목록, 단계별 조리 순서
- 조회수(일별 관리), 좋아요, 북마크
- 댓글 작성/삭제 및 별점 평가
- 조리 전용 모드 진입 버튼

### 2-3. 레시피 직접 작성
- 텍스트 입력 + 이미지 업로드 (Firebase Storage)
- 단계별 조리 순서 추가/삭제
- 드래프트(임시 저장) 기능 (SharedPreferences)

### 2-4. 조리 전용 모드
- 풀스크린, 큰 글씨 표시
- 단계별 페이지 뷰 (좌우 스와이프)
- 진행률 표시

---

## 3. 개인화 기능

- **북마크** — 레시피 저장, 북마크 목록 화면 별도 제공
- **최근 본 레시피** — 자동 기록, 최근 순 목록 제공
- **좋아요** — 레시피 및 커뮤니티 게시글 좋아요
- **활동 내역** — 레시피 댓글, 커뮤니티 댓글, 커뮤니티 게시글, 작성한 레시피 4탭

---

## 4. 냉장고 기능

- **재료 등록** — 자주 쓰는 재료 칩(빠른 선택) + 직접 입력
- **유통기한 관리** — 날짜 설정, 임박/만료 시 알림 발송
- **냉장고 파먹기** — 보유 재료 선택 → 만들 수 있는 레시피 자동 매칭
  - 난이도 필터, 조리 시간 필터 추가 지원

---

## 5. 커뮤니티 기능

### 5-1. 게시글
- **카테고리:** 전체, 공지(관리자 전용), 자유, Q&A, 나눔
- 이미지 첨부, 글 작성/수정/삭제
- 조회수 자동 증가, 좋아요

### 5-2. 댓글
- 커뮤니티 게시글·레시피 모두 댓글 가능
- 작성/삭제, 작성자 프로필 이미지 표시

### 5-3. 1:1 채팅(쪽지)
- 게시글/레시피 컨텍스트 기반 채팅방 생성
- 채팅 목록 화면 (최신순, 마지막 메시지 미리보기)

### 5-4. 사용자 프로필
- 프로필 공개/비공개 설정 (프로필, 레시피, 게시글 각각)
- 타 사용자의 작성 레시피·게시글 탭 조회

## 6. 알림 기능

| 알림 타입 | 발생 조건 |
|-----------|----------|
| 레시피 댓글 알림 (`recipe_comment`) | 내 레시피에 새 댓글이 달렸을 때 |
| 레시피 좋아요 알림 (`recipe_like`) | 내 레시피에 좋아요가 눌렸을 때 (본인 제외) |
| 인기 순위 진입 알림 (`trending`) | 내 레시피가 오늘 TOP 10에 진입했을 때 (하루 1회) |
| 답글 알림 (`recipe_reply`) | 내 댓글에 답글이 달렸을 때 |
| 커뮤니티 댓글 알림 (`community_comment`) | 내 게시글에 새 댓글이 달렸을 때 |
| 유통기한 임박 알림 (`fridge_expiry`) | 냉장고 재료 유통기한 D-3 이하 |
| 유통기한 만료 알림 (`fridge_expiry`) | 냉장고 재료 유통기한 초과 |

- 알림 개별 읽음 / 모두 읽음 처리
- 알림 클릭 시 해당 게시글·레시피로 이동
- 설정에서 알림 종류별 ON/OFF 가능 (새 레시피 / 커뮤니티 댓글 / 유통기한 임박)

## 7. 업적 & 칭호 시스템

총 **26가지 칭호**, **7개 카테고리** 기반 달성도 표시

| 카테고리 | 칭호 예시 | 조건 |
|----------|-----------|------|
| 레시피 탐험가 | 식탐러 → 전설의 미식가 | 레시피 조회 10~300회 |
| 즐겨찾기 수집가 | 메모장 → 레시피 도서관 | 북마크 5~100개 |
| 레시피 평론가 | 맛 초보 → 식신 | 댓글 3~50개 |
| 커뮤니티 주민 | 새내기 → 터줏대감 | 게시글 1~30개 |
| 레시피 창작자 | 견습생 → 미슐랭 셰프 | 레시피 작성 1~20개 |
| 세계 요리 탐방 | 동네 미식가 → 세계 미식 대가 | 국가별 레시피 탐방 3~7개국 |
| 냉장고 파먹기 | 냉장고 청소부 → 재료 연금술사 | 냉장고 검색 5~50회 |

**특별 업적:**
- 얼리버드: 특정 시기 가입한 사용자
- 완벽주의자: 모든 카테고리에서 최소 1개 이상 달성
- 전설: 26가지 칭호 모두 달성

## 8. 관리자 기능

관리자 이메일 계정으로 로그인 시 관리자 패널 접근 가능

| 탭 | 기능 |
|----|------|
| 대시보드 | 전체 사용자·레시피·게시글 수 통계 |
| 레시피 관리 | 검색, 삭제, 공공 API 데이터 동기화 |
| 커뮤니티 관리 | 게시글·댓글 검색 및 삭제 |
| 유저 관리 | 사용자 검색, 프로필 조회, 삭제 |

- 관리자만 커뮤니티에 **공지** 카테고리 게시글 작성 가능
- 관리자에게는 자동으로 **운영자** 칭호 부여
