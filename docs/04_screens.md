# 주요 화면 상세 설명

## 1. 로그인 / 회원가입

### LoginScreen
- Google, 카카오, 이메일 3가지 로그인 방식 제공
- 이전 로그인 방식 힌트 자동 표시 (SharedPreferences)
- **비로그인 게스트 모드** 지원 — 로그인 없이도 앱 진입 가능. 마이페이지에서 "게스트"로 표시되며, 북마크·활동 내역·채팅 등 로그인 필요 기능 탭 시 스낵바로 로그인 유도

### SignupScreen
- 이메일·비밀번호·닉네임 입력
- 닉네임·이메일 실시간 중복 체크 (Firestore)
- 회원가입 성공 후 로그인 화면으로 이동

### ForgotPasswordScreen
- 이메일 입력 → Firebase 비밀번호 재설정 메일 발송

### TermsAgreementScreen
- 이용약관·개인정보처리방침 동의 체크 후 회원가입 진행

## 2. 홈 (HomeScreen)

| 구성 요소 | 내용 |
|-----------|------|
| 자동 회전 배너 | 누적 조회수 TOP 4 레시피 슬라이드 (4초 간격 자동 전환) |
| 냉장고 배너 | 나만의 냉장고 / 냉장고 파먹기 빠른 진입 버튼 2종 |
| 카테고리 그리드 | 국가별 요리 카테고리 7종 아이콘 탭 |
| 🔥 트렌딩 | 좋아요×3 + 조회수 합산 기준 인기 레시피 TOP 6 |
| ⭐ 오늘의 추천 | 오늘 조회수 TOP 3 레시피 (피처드 대형 카드 + 소형 카드) |
| ❤️ 좋아요 TOP | 전체 좋아요 수 상위 6개 레시피 |
| 📅 어제 많이 본 | 어제 인기 급상승 레시피 TOP 6 |

- 반응형 레이아웃 — 화면 너비 700px 이상이면 카드 격자(Wrap), 미만이면 가로 스크롤 리스트로 자동 전환

## 3. 레시피 (RecipeListScreen)

### 레시피 목록
- **국가 필터:** 한식, 서양, 중국, 일본, 동남아시아, 이탈리아, 퓨전
- **유형 필터:** 기타, 반찬 등
- **정렬:** 최신순, 인기순
- 무한 스크롤로 레시피 카드 표시

### RecipeDetailScreen (레시피 상세)
- 대표 이미지, 칼로리, 분량, 조리 시간, 난이도 표시
- 재료 목록 (재료명 + 용량)
- 단계별 조리 순서 + 단계별 이미지(`stepImages`) 지원
- 조회수 (일별 관리: `todayDate` 비교), 좋아요, 북마크 버튼
- 인기 순위 TOP 10 진입 시 작성자에게 알림 자동 발송 (하루 1회)
- **유저 작성 레시피(`source: ugc`)** 별도 표시, 작성자 프로필 카드 노출 및 채팅 버튼 제공
- **조리 전용 모드** 버튼
- 반응형 레이아웃 — 900px 이상 시 레시피 정보(좌) + 댓글(우) 2컬럼 배치

**댓글 세부 기능:**
- 별점(1~5점) 선택 후 댓글 작성, 댓글 목록 상단에 평균 별점 표시
- 익명 / 닉네임 전환 토글 (로그인 사용자만)
- 댓글 좋아요 (로그인 사용자만, `commentLikes` 서브컬렉션에 저장)
- 댓글 수정 / 삭제 (본인 댓글 또는 레시피 작성자)
- **대댓글(답글)** 작성·수정·삭제, 답글 수 표시 및 펼치기/숨기기
- 댓글 작성자에게 **쪽지 보내기** (PopupMenu → `ChatScreen` 이동)

### CookingModeScreen (조리 전용 모드)
- 풀스크린 전용 화면, 큰 글씨
- 단계별 페이지 뷰 (좌우 스와이프)
- 상단 진행률 바 표시

### RecipeCreateScreen (레시피 작성)
- 제목, 소개글, 재료 목록, 단계별 조리 순서 입력
- 대표 이미지 업로드 (Firebase Storage)
- 드래프트(임시 저장) 자동 보관 (SharedPreferences)

## 4. 냉장고 & 냉장고 파먹기

### FridgeScreen (냉장고 관리)
- 자주 쓰는 재료 칩으로 빠르게 추가
- 재료명·수량·유통기한 직접 입력
- 재료 삭제
- 유통기한 임박(D-3 이하) 시 알림 발송 → `NotificationsScreen`

### IngredientSearchScreen (냉장고 파먹기)
- 냉장고 재료 목록에서 사용할 재료 선택
- 선택한 재료를 포함하는 레시피 자동 매칭 (클라이언트 필터링)
- 난이도·조리 시간 필터 추가 지원

---

## 5. 커뮤니티 (CommunityScreen)

### 게시글 목록
- **카테고리 탭:** 전체 / 공지(관리자만) / 자유 / Q&A / 나눔
- FAB 버튼으로 게시글 작성
- 카드 형태로 제목, 작성자, 조회수, 좋아요 수, 댓글 수 표시

### CommunityPostDetailScreen (게시글 상세)
- 게시글 내용, 첨부 이미지 표시
- 조회수 자동 증가, 좋아요 버튼
- 댓글 작성/삭제
- 작성자 프로필 → `UserProfileScreen` 이동
- **채팅 시작** 버튼 → `ChatScreen` 이동

### ChatListScreen / ChatScreen (1:1 채팅)
- 채팅방 목록 최신순 정렬, 마지막 메시지 미리보기
- 게시글/레시피 컨텍스트 기반 채팅방 자동 생성
  - 채팅방 ID: `{user1_id}_{user2_id}_{contextId}`

### UserProfileScreen (사용자 프로필)
- 닉네임, 프로필 이미지, 칭호 표시
- 작성 레시피 탭, 커뮤니티 게시글 탭
- 비공개 설정 시 목록 숨김

## 6. 마이페이지 (MyPageScreen)

| 섹션 | 메뉴 항목 | 이동 화면 |
|------|-----------|----------|
| 나만의 냉장고 | 재료 관리 (유통기한 알림) | FridgeScreen |
| 내 활동 | 즐겨찾기(북마크) | BookmarksScreen |
| 내 활동 | 최근 본 레시피 | RecentRecipesScreen |
| 내 활동 | 내 활동 내역 | MyActivityScreen |
| 내 활동 | 채팅 | ChatListScreen |
| 설정 | 알림 설정 | 다이얼로그 (새 레시피/커뮤니티 댓글/유통기한 알림 ON/OFF) |
| 설정 | 개인정보 처리방침 | PrivacyPolicyScreen |
| 설정 | 앱 정보 | AppInfoScreen — 버전, 주요 기능 소개, 고객지원 이메일, 이용약관(TermsScreen), 개인정보 처리방침, 오픈소스 라이선스(OpenSourceLicenseScreen) |
| 설정 | 관리자 패널 (관리자만) | AdminScreen |
| — | 로그아웃 | LoginScreen |

### EditProfileScreen (프로필 수정)
- 닉네임 변경 (중복 체크 포함)
- 프로필 사진 업로드/변경 (Firebase Storage)
- 획득한 칭호 중 대표 칭호 선택
- 프로필·레시피·게시글 공개 여부 개별 설정

### MyActivityScreen (활동 내역)
- 4개 탭: 레시피 댓글, 커뮤니티 댓글, 커뮤니티 게시글, 작성한 레시피

### AchievementsScreen (업적/칭호)
- 26가지 칭호 달성 현황
- 카테고리별 진행도 표시 (예: 레시피 10/50/100/300회)
- 특별 업적 3종: 얼리버드, 완벽주의자, 전설

### TermsScreen (이용약관 전문)
- 앱 정보 → 이용약관 탭으로 진입
- 제1조(목적) ~ 제8조(준거법 및 분쟁 해결) 전문 표시
- 시행일: 2026년 6월 1일

### OpenSourceLicenseScreen (오픈소스 라이선스)
- 앱 정보 → 오픈소스 라이선스 탭으로 진입
- 사용 패키지 12종을 라이선스 종류(BSD 3-Clause / Apache 2.0 / MIT)별로 분류
- ExpansionTile로 각 패키지 용도·pub.dev 링크 확인 가능

---

## 7. 알림함 (NotificationsScreen)

- 알림 최근 50개를 실시간 스트림으로 표시
- 알림 종류: 레시피 댓글, 커뮤니티 댓글, 유통기한 임박/만료
- 개별 읽음 처리, 모두 읽음 버튼
- 알림 클릭 → 해당 레시피·게시글로 이동

---

## 8. 관리자 패널 (AdminScreen)

관리자 이메일로 로그인 시에만 마이페이지에 관리자 패널 메뉴 노출

| 탭 | 주요 기능 |
|----|----------|
| 대시보드 | 전체 사용자·레시피·게시글 수 실시간 통계 |
| 레시피 관리 | 레시피 검색·삭제, 공공 API 데이터 동기화 |
| 커뮤니티 관리 | 게시글·댓글 검색·삭제 |
| 유저 관리 | 사용자 검색·프로필 조회·삭제 |
