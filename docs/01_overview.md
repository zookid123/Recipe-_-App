# 프로젝트 개요

## 앱 소개

**냉장고 구조대**는 농림축산식품부 공공 레시피 API를 기반으로 한 소셜 요리 플랫폼입니다.
단순한 레시피 조회를 넘어 커뮤니티, 냉장고 관리, 업적 시스템까지 갖춘 종합 요리 앱입니다.

---

## 기획 의도

- 공공 데이터(농림축산식품부 레시피 API)를 활용해 신뢰도 높은 레시피 제공
- 사용자들이 직접 레시피를 작성·공유하며 커뮤니티를 형성
- 냉장고에 있는 재료를 입력하면 만들 수 있는 레시피를 추천하는 "냉장고 파먹기" 기능으로 음식 낭비 최소화
- 업적·칭호 시스템으로 지속적인 앱 사용 동기 부여

## 핵심 사용자 시나리오

### 시나리오 1: 자취생의 냉장고 파먹기

> 냉장고에 계란, 대파, 김치만 남은 자취생이 저녁 메뉴를 찾는다.

1. 앱 실행 → Google 로그인
2. 홈 화면에서 **냉장고 배너** 탭 → 냉장고 파먹기 진입
3. 보유 재료 칩에서 `계란`, `대파`, `김치` 선택
4. 매칭된 레시피 목록 확인 → 조리 시간 30분 이하 필터 적용
5. **김치볶음밥** 레시피 선택 → 재료 목록·단계별 조리 순서 확인
6. **조리 전용 모드** 진입 → 풀스크린 큰 글씨로 단계별 요리

**핵심 검증 포인트**: 재료 매칭 정확도, 필터 동작, 조리 전용 모드 전환

---

### 시나리오 2: 요리 마니아의 레시피 공유

> 직접 만든 레시피를 앱에 올리고 커뮤니티에서 반응을 확인한다.

1. 레시피 탭 → **+ 버튼** → 레시피 작성 화면 진입
2. 제목·소개글·재료 목록·단계별 조리 순서 입력 + 대표 이미지 업로드
3. 임시 저장(드래프트) 후 완성 → 레시피 게시
4. 커뮤니티 탭 → **자유** 카테고리에 요리 자랑 게시글 작성 + 이미지 첨부
5. 다른 사용자 댓글·좋아요 수신 → 알림함에서 확인
6. 레시피 조회수 TOP 10 진입 → **인기 순위 진입 알림** 수신
7. 업적 화면에서 **레시피 창작자** 칭호 달성 확인

**핵심 검증 포인트**: 이미지 업로드, 알림 발송, 업적 달성 트리거

---

### 시나리오 3: 다이어터의 레시피 탐색

> 칼로리와 조리 시간을 기준으로 건강한 레시피를 찾아 북마크한다.

1. 레시피 탭 → 국가 필터 **한식** 선택 → 정렬 **인기순** 적용
2. 조리 시간 25분 이하 레시피만 확인 (홈 화면 ⚡ 초간단 레시피 섹션 활용)
3. 칼로리 낮은 레시피 상세 진입 → 재료·칼로리 확인
4. **북마크** 버튼 탭 → 마이페이지 즐겨찾기에 저장
5. 냉장고에 재료 추가 + 유통기한 설정 → D-3 임박 시 알림 수신

**핵심 검증 포인트**: 필터·정렬 조합, 북마크 저장, 유통기한 알림
